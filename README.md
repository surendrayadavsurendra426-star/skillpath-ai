# 🎯 SkillPath AI Pro
**A Personalized AI Mentor for Your Career Journey**

Developed by: **Mr. Surendra Yadav** 🚀

## ✨ Features
- 🎙️ **Voice Recognition:** Speak your skill name to generate a roadmap.
- 🗺️ **30-Day Roadmaps:** Structured day-by-day learning plans.
- 📄 **Professional PDF:** Download your roadmap in a clean PDF format.
- 🌍 **Multilingual:** Supports Hinglish, Hindi, and English.
- 📲 **WhatsApp Share:** Share your growth plan with friends instantly.
- 📺 **YouTube Integration:** Direct links to the best tutorials.

## 🛠️ Tech Stack
- **Python** (Streamlit, Google Gemini AI, FPDF)
